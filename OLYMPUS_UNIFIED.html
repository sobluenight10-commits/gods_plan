📜 §CODEX — HOW TO READ OLYMPUS — every index, definition, interpretation, implied action · click to expand
① GEM GRADE (A / B+ / B / C)
What: Monte-Carlo Heston projection ranking a stock by 1-5yr expected value distribution.
Read: A = top-quartile asymmetry · B+ = above-median edge · B = edge thinning · C = negative expected value.
1Y EV = 1-year expected return (median of sim paths). 5Y EV = 5-year compounded EV.
Use: A-grades → candidate for ADD. B-grades with negative 1Y EV → TRIM watchlist. Downgrade A→B in one day → investigate the driver, don't auto-sell.
② CATALYST RADAR — ASYMMETRY SCORE (−100 to +100)
What: Combines historical base rate (last 4-8 similar events) with forward signals (analyst tone, static event direction).
Read: >+40 skew upside · ±20 symmetric · <−40 skew downside.
Use: Positive asymmetry + UNDER_POSITIONED flag = pre-event add. Negative asymmetry + ELEVATED_EXPECTATIONS = trim before print.
③ CATALYST RADAR — BASE RATE
What: Historical T−1 to T+1 close move across last 4-8 earnings.
Shows: mean_abs_move · win_rate · best/worst pct.
Use: If base-rate win_rate >70% AND current <50d MA → high-conviction pre-event add. If win_rate <40% AND current >50d MA → skip or trim.
④ CATALYST RADAR — SETUP FLAG
UNDER_POSITIONED = quiet attention + below MA → buy setup.
ELEVATED_EXPECTATIONS = high attention + above MA → fade setup.
QUIET_PRE_EVENT = no attention spike yet → watch.
CROWDED = retail attention percentile >90 → reduce.
THESIS_RISK = thesis is WOUNDED/DEAD → do nothing additive.
⑤ ATTENTION SIGNAL (0-100)
What: Google Trends retail search interest for the ticker, normalized over 2-year baseline.
Read: 0-30 ignored · 30-60 normal · 60-85 elevated · 85+ parabolic retail chase.
Use: Attention ≥85 + fundamentals unchanged = fade the crowd. Attention ≤30 on high-conviction name = institutional window.
⑥ NET LIQUIDITY (§11)
Formula: Reserves − TGA − RRP (all FRED, weekly).
Zones: DANGER <$2.0T (SVB corridor) · WARNING $2.0-2.5T · NORMAL $2.5-3.5T · ABUNDANCE >$3.5T.
Direction: EXPANDING ↑ = deploy window. CONTRACTING ↓ = reduce.
Corridor: ABOVE / INSIDE / BELOW the rolling p25-p75 band. BELOW = liquidity shock, risk-off. ABOVE = fuel for rally.
Regime: tight / neutral / loose (GEM feeds from this).
Rule: When DANGER + CONTRACTING + BELOW corridor → freeze all adds regardless of individual GEM grades.
⑦ THESIS STATE (guard-enforced)
INTACT = original reason still valid, no disqualifying event.
WOUNDED = partial invalidation (competitive loss, guidance cut, key exec exit, regulatory setback).
DEAD = bankruptcy, fraud, total TAM collapse, competitor kill-shot.
Invalid triggers (blocked): price_dropping, analyst_downgrade, valuation_too_high, sector_peers_dropping, technical_breakdown.
Use: Only DEAD → full EXIT. WOUNDED → TRIM and observe. INTACT + price drop → this is an ADD candidate, not a panic.
⑧ DECISION LEDGER (FEEL / LEARN / CONVICTION / HORIZON)
FEEL = free text captured at decision time (FOMO, fear, doubt, conviction, regret, greed).
LEARN = post-hoc lesson from the outcome.
CONVICTION = 1-5, audited by weekly review vs hit rate.
HORIZON = expected holding period (1-3mo / 6-12mo / multi-year).
Use: Sunday 09:00 Berlin — paste Keep notes, ledger_ingest.py parses them. Weekly review shows which emotional state wins and loses.
⑨ PRE-ALERT (Telegram)
STOCKS = tickers extracted from blog/news themes.
ESTIMATES = expected direction + magnitude if catalyst confirms.
TIMING = ACT / WATCH / SKIP.
Rule: ACT only when STOCKS ≠ NONE AND ESTIMATES has numbers. Otherwise WATCH.
⑩ PRIORITY OF SIGNALS (when conflict)
1. THESIS GUARD — overrides everything. DEAD = exit regardless.
2. LIQUIDITY ZONE — DANGER freezes adds.
3. CATALYST RADAR — sets timing (pre-event setup).
4. GEM GRADE — sets position sizing.
5. ATTENTION + PRICE vs MA — sets entry price discipline.
If any layer contradicts the one below, the higher layer wins.

Strike Score vs GEM Grade — why they tell different stories
GEM Grade (the bottom table on the page) measures only the shape of the projected return distribution — Heston Monte Carlo EV vs ES5 vs 9-dimension fundamental risk. It answers: "if I held this for 1y/5y, how good is the math?"
Strike Score (this panel) is a deployment-context composite: it takes that same GEM forecast (20% weight) and combines it with macro fit (25%), peer-relative valuation/OPS (15%), conviction/GOD score (15%), nearest catalyst (10%) and discovery edge (10%), then subtracts risk penalties. It answers: "of the names I could buy with my powder TODAY, which is most asymmetric right now?"

Common reasons they diverge:
  • High GEM, low Strike — great math but expensive (OPS ≥ 150) or in a freeze regime. Example: PLTR (A grade but OPS ~300 → premium decoupled).
  • Low GEM, high Strike — modest math but cheap, near catalyst, fits current macro pivot. Example: a CORE name with B grade but OPS < 50 in a STRIKE_PIVOT_EARLY state.
  • Both high — converged signal. These are the deploy candidates (000660.KS, TSM today).
Use both: GEM Grade for the long-hold decision, Strike Score for the deployment decision. They are complementary — not contradictory.
